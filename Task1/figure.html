Figure

Элемент figure представляет собой самодостаточный блок контента с возможным заголовком, на который, как правило, ссылаются из основного потока документа. Этот блок может быть перемещён из основного потока документа без потери общего смысла документа.Например:

Figcaption

Элемент figcaption представляет собой заголовок или легенду для figure. Элемент figcaption не обязателен и добавляется до или после содержимого внутри figure. Только один figcaption может находиться внутри figure, при этом допустимо добавлять множество других дочерних элементов (например, img или code).Например:

Плюсы и минусы использования этих тегов:

  1. figure исользуется для
    • блоков кода
    • аудио
    • видео
    • рекламных объявлений
    • Можно вложить один блок figure в другой
  2. figcaption
    • Не каждый тег figure нуждается в figcaption
    • не обязателен
    • добавляется до или после содержимого внутри figure
    • только один figcaption может находиться внутри figure
